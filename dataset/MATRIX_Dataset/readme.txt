Methane Release Experiment (MATRIX) Data

Data during the whole campaign is available in each responding folder with their corresponding flight codes. Files with names starting with "EMP" are measurements obtained using Empa's Quantum Cascade Laser (QCLAS). Files with names starting with "RUG" are measurements obtained from Groningen University's (RUG) Active AirCore system. Post-processed meteorology files are availale for each day and are located in the folder named "MATRIX_Meteo".
