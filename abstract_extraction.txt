1.3 Fundamental Scientific and Engineering Concepts 
This project is based on a combination of fundamental scientific principles and core 
engineering concepts drawn from petroleum engineering, electrical engineering, and computer 
science. These concepts collectively enable the detection of methane leaks, the conversion of 
physical phenomena into measurable electrical signals, and the analysis of sensor data within a 
controlled laboratory environment. 
1.3.1  Infrared Absorption Principle for Methane Detection 
The fundamental scientific principle underlying the proposed methane detection 
system is infrared (IR) absorption. Methane molecules absorb infrared radiation at specific 
wavelengths due to molecular vibrational modes. When infrared light propagates through a 
region containing methane gas, a portion of the incident radiation is absorbed, resulting in a 
reduction in the transmitted light intensity. 
This absorption behavior is described by the Beer–Lambert principle, which relates the amount 
of absorbed radiation to the gas concentration and the optical path length. The conceptual 
behavior of this relationship is illustrated in Figure 1, where infrared radiation emitted from a

detecting relative changes in transmitted signal intensity, which is sufficient for identifying leak 
events under controlled laboratory conditions. 
From a petroleum engineering perspective, this principle supports the analysis of methane

1.3.2  Infrared Sensing Configuration 
To implement the infrared absorption principle, the system uses an infrared emitter–
detector pair positioned across selected regions of the pipeline test section. The emitter 
generates continuous infrared radiation directed toward the detector. Under normal operating 
conditions, the detector receives a stable signal corresponding to unobstructed transmission. 
When methane leaks into the optical path, partial absorption of infrared radiation occurs, 
leading to a reduction in the detected signal intensity. This configuration represents a non-
dispersive infrared sensing approach, which is suitable for bench-top experimentation due to 
its simplicity, robustness, and repeatability. Mechanical alignment, optical shielding, and fixed 
geometry are used to minimize the influence of ambient light and environmental disturbances. 
 
1.3.3  Signal Conditioning and Data Acquisition 
From an electrical engineering standpoint, the infrared detector produces a low-
amplitude analog electrical signal that is susceptible to noise and interference. To ensure 
reliable measurement, signal conditioning is applied before digital processing. This includes 
amplification to increase signal magnitude and filtering to suppress unwanted noise 
components. 
The conditioned signal is then digitized using an analog-to-digital converter (ADC) integrated 
within a microcontroller. Digitization enables consistent sampling, numerical representation of 
the sensor output, and time-stamping of measurements. These steps are essential for reliable 
data transmission and subsequent software-based analysis. 
In the final system design, digitized measurements are structured into time-stamped data frames 
and transmitted securely to the backend using HTTPS, ensuring encrypted communication 
between the embedded system and cloud infrastructure.

 The microcontroller functions as the central embedded system, managing sensor sampling, 
local data buffering for time-series continuity, and secure communication with external

1.3.4  Data Transmission and Cloud-Based Monitoring 
 Communication engineering concepts are applied through secure HTTPS-based data 
transmission from the embedded system to the backend cloud platform.  All transmitted data is 
secured using Transport Layer Security (TLS), which performs authenticated key exchange and 
encrypted communication using AES-based symmetric encryption after session establishment. 
This architecture reflects modern monitoring system designs used in industrial and research 
applications. 
 The FastAPI backend and cloud database serves as an intermediate layer that allows sensor 
data to be accessed by higher-level software applications without direct physical interaction 
with the hardware. 
 
1.3.5  Data-Driven Analysis and Artificial Intelligence 
From a computer science perspective, the project employs data-driven analysis 
techniques to support methane leak detection. Rather than relying solely on fixed threshold 
values, sensor data is analyzed to identify patterns associated with normal operation and leak 
events. Features such as signal magnitude variation and temporal behavior are extracted from 
the infrared measurements. 
 An unsupervised anomaly detection approach based on the Isolation Forest algorithm is 
applied to model normal operating behavior and detect deviations indicative of methane 
leakage. Instead of relying on labeled leak data, the model learns baseline signal patterns and 
assigns an anomaly score to new observations based on structural deviation in the feature space. 
The role of artificial intelligence in this project is  to provide predictive anomaly scoring and

1.3.6  System-Level Integration 
The integration of infrared sensing, signal conditioning, embedded processing, wireless 
communication, and data-driven analysis forms a complete methane leak detection system. 
Each subsystem performs a specific function, and overall performance depends on their 
coordinated operation. Real-time inference within the backend ensures that anomaly detection 
results are generated within the defined system response constraint, while secure transmission 
and structured data handling maintain reliability across subsystem boundaries.   From a systems 
engineering perspective, the project emphasizes integration and experimental validation rather 
than component-level optimization. 
By combining petroleum engineering context, electrical sensing hardware, and computer 
science–based data analysis, the project provides a multidisciplinary platform for evaluating 
methane leak detection behavior, response time, and reliability under controlled laboratory 
conditions. 
 
1.3.7  Summary 
The application of infrared absorption physics, basic signal processing, embedded 
system design, wireless communication, and data-driven predictive anomaly detection forms 
the scientific and engineering foundation of the proposed methane detection system. These 
concepts collectively enable the implementation and evaluation of a bench-top prototype that 
supports controlled testing, performance assessment, and future scalability while remaining 
aligned with academic design requirements and project scope.
